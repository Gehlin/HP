# Phase 03: App-Wide Propagation & Consistency Sweep

The app is much bigger than the five core screens — it has subject hub pages (Verbal, Quant), a dozen theory guides, an SRS review queue, bookmarks, an Ord builder, a score predictor, the exam-select/start flow, and shared components like `ExplanationCard`, `AchievementToast`, `FormulaDrawer`, and `ChartView`. Most of these were already converted away from the old dark theme in earlier work, but they may still reference pre-Phase-01 token values, inconsistent card patterns, or component states that don't yet use the richer answer-state/feedback palette added in Phase 01. This phase sweeps every remaining page and shared component so the entire app — not just the five mockup screens — feels like one coherent system, and removes any duplicated or inconsistent card/button/badge styling along the way.
