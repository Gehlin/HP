## Hardcoded Hex Colors Needing Token Replacement

### Theory.tsx — Many hardcoded hex values
- `#2563EB` → use `var(--color-green)` or `var(--color-terracotta)` for section accent (blue is off-system)
- `#7C3AED`, `#9333EA` → violet — no matching token; use `var(--color-terracotta)` for verbal accent
- `#DC2626`, `#DB2777` → red/pink — no matching token; verbal accent → `var(--color-terracotta)`
- `#D97706` → amber — closest is `var(--color-gold)` for "trap/fälla" callout
- Subject dot colors (span.rounded-full): map each subject to its token accent

### Progress.tsx
- `#224A3A` (= `--color-green`) used raw in heatmap legend divs — wrap in `var(--color-green)`

### ChartView.tsx
- `#9A9A92` → `var(--color-ink-faint)` or `var(--color-muted)`
