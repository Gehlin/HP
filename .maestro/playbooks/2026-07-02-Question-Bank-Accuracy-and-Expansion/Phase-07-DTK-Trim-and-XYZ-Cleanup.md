# Phase 07: DTK Column Trim & XYZ Option-Count Cleanup

Two independent, user-confirmed, purely mechanical fixes from Phase 06's audit (see `Reference/audit-findings.md` for full detail):

1. **DTK mobile clipping:** 11 existing questions (`dtk-031`–`035`, `dtk-036`–`040`, `dtk-e14`) have tables wide enough to clip on mobile viewports, hiding data — in 2 cases (`dtk-036`–`040`, `dtk-e14`) the clipped column is the one the question actually depends on. Fix: trim each table to drop a column the question doesn't need, matching the precedent already used successfully in Phase 05.
2. **XYZ option-count cleanup:** 55 items (`xyz-e21`–`xyz-e75`) have 5 answer options where the real exam (and the rest of the app's XYZ bank) uses 4. Fix: remove the extra option from each, keeping the 4 most useful (the correct answer + the 3 strongest distractors).
